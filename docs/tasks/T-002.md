# T-002 CSV Upload, Profiling, and Routing

## Scope

- CSV upload from UI
- Backend CSV parsing
- Dataset profile response
- Sales/generic route response
- Profile/report shell UI
- Focused backend tests

## Acceptance

- UI can upload a CSV.
- API returns row count, column count, inferred types, missing percentages, duplicate rows, sample rows, and warnings.
- API returns dataset type, confidence, and explanation.
- Tests cover profiler, router, and upload integration.
